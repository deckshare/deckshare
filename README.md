# Deckshare

A trading card game database and platform for deck building and sharing.

## Development

### Requirements

- Ruby (see [`.ruby-version`](.ruby-version))
- Node (or compatible runtime)
- PostgreSQL 16
- OpenSearch 2

### Setup

```sh
bin/setup
```

### Running

```sh
bin/dev
```

or run the "Rails server" debug configuration in Visual Studio Code.
